# Security Policy for Isuzu Manila Sales Site

## Supported Versions
We currently support and provide updates for the following version:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability
If you discover a security vulnerability or a technical bug on this website, please do not report it through public issues. 

Instead, please contact the site administrator directly via the contact details provided on **isuzu-manila.uk**. 

We aim to acknowledge all reports within 48 hours and resolve any critical issues as quickly as possible to ensure our customers have a safe browsing experience.
